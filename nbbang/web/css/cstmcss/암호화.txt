암호화
 1. 단방향 암호화: 한 번 암호화하면 복호화(해독) 불가능한 암호화 방식. -> 비밀번호 SHA512 방식
 
  2. 양방향 암호화: 암호화, 복호화가 가능한 방식.
     암호화 할때 key를 이용하게 됨! 암호화, 복호화 할 수 있는 key!
     AES: key 한 개로 암호화, 복호화를 하는 것. ->key 배분, 관리하기 어려움.
     RSA: key를 두 가지 가지고 있음. 
	o 개인키: 암호화를 하는 키, 나만가지고 있음.
	o 공용키: 복호화를 하는 키, 누구든지 가져올 수 있음.  


암호화에 대해 기본적으로 JAVA에서 제공하는 api를 사용하면 됨.
